## Installation
__Requirements__

sharpy is developed and tested using:
+ Anaconda Python 3.6
+ GCC 5.4.0 or 6
+ CentOS 7 and MacOS Sierra

Certain dependencies are required separately:
+ [LAPACK](http://www.netlib.org/lapack/)
+ [Eigen](http://eigen.tuxfamily.org)
