Pre-Sharpy¶

class sharpy.presharpy.presharpy.PreSharpy(in_settings)[source]¶

The main class for preSHARPy.

PreSharpy objects contain information on the output of the problem. The PreSharpy settings attributes are obtained from the .solver.txt file, under [SHARPy].

Parameters:in_settings – settings file, from which [SHARPy] values are extracted
settings¶

name-value pairs of solution settings types. See Notes for acceptable combinations.

Type:dict
settings_types¶

accepted types for values in settings

Type:dict
settings_default¶

default values for settings, should none be provided.

Type:dict
ts¶

solution time step

Type:int
case_route¶

route to case folder

Type:str
case_name¶

name of the case

Type:str

Notes

The following key-value pairs are acceptable settings for the PreSharpy class:

Key Type Description Default
flow list(str) List of solvers to run in the appropriate order None
case str Case name default_case_name
route str Route to folder None
write_screen bool Write output to terminal screen True
write_log bool Write log file to output folder False
log_folder str Folder to write log file within /output None
log_file str Log file name log
